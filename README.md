## Overview 
Vyapaar Connect is a mobile-first prototype designed to help small vendors digitize customer engagement and loyalty tracking through a simple and structured workflow. 

## Core Features 
- Sellers upload and manage menu or product details via a Flutter-based mobile interface 
- Image capture with AI-assisted text extraction (~66% accuracy during initial prototype testing) 
- Digital transaction recording at the time of purchase 
- QR code generation for customer interaction 
- Web interface for loyalty point tracking and redemption 

## Development Approach 
- Product-driven development 
- AI-assisted development workflow 
- Rapid prototyping under competitive presentation timelines
